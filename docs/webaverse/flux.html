    FLUX

    FLUX is a token that is used in Webaverse primarily to mint NFTs, but also can function as a currency to trade with.

    FAQ

    How Can I Buy FLUX?

    You can buy FLUX on Uniswap by swapping for ETH. If you want a step-by-step guide on how to buy FLUX on Uniswap, check out this guide.

    Technically, what is FLUX?

    FLUX is an ERC-20 token based on the Ethereum ERC-20 Token Standard. This means you can own and manage your FLUX tokens the same way you would other Ethereum based tokens like ETH.
